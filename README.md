## **5. Setting up our project & directory explained** 
Creating a new Symfony Project:

    composer create-project symfony/skeleton *project name*
or

    composer symfony/(website-)skeleton *project name*

Project tree:
* bin
* config
* public (root of our dir)
* src (code right here)
* var (cache and logs)
* vendor (packages)
